# Cinnex-Dev org-wide GitHub defaults

This repository holds default community health files for the [Cinnex-Dev](https://github.com/Cinnex-Dev) organisation.

Any Cinnex-Dev repo that does **not** have its own `.github/ISSUE_TEMPLATE/` or `.github/pull_request_template.md` will automatically inherit the templates here.

## Contents

| Path | Purpose |
|---|---|
| `.github/ISSUE_TEMPLATE/bug.yml` | Bug report form |
| `.github/ISSUE_TEMPLATE/feature.yml` | Feature request form |
| `.github/ISSUE_TEMPLATE/chore.yml` | Chore / infra work form |
| `.github/ISSUE_TEMPLATE/config.yml` | Disables blank issues |
| `.github/pull_request_template.md` | Default PR template |
| `CONTRIBUTING.md` | Comment voice rules + lint script instructions |

## Updating templates

Edit the files in this repo. Changes take effect immediately for all repos that inherit these defaults. Repos with their own `.github/` templates are unaffected — update those separately.
